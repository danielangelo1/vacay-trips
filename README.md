## 💻 Getting started

### Requirements


**Clone the project and access the folder**

```bash
$ git clone https://github.com/danielangelo1/vacay-trips.git && cd vacay-trips
```

**Follow the steps below**

```bash
# Install the dependencies
$ yarn


# Start the client
$ yarn dev
```
